## Psychology Principles Applied

| Principle | Where Used | Effect |
|-----------|-----------|--------|
| **Reciprocity** | Welcome emails give value first | Creates obligation to engage |
| **Commitment & Consistency** | Small asks escalate gradually | Each step builds toward partnership |
| **Social Proof** | Mention network size/quality | Builds trust through belonging |
| **Authority** | Professional tone, clear process | Positions Jackie as credible |
| **Scarcity** | VIP/priority access mentions | Motivates action |
| **Peak-End Rule** | Strong openings + warm closes | Memorable impressions |
